# Jin Blog

一个面向公开展示的个人博客，主打「年-月-周」事件记录，同时支持复盘、观点文章、标签与搜索，整体视觉参考 Apple 官网的简洁科技风格。

- 结构化记录（Year → Month → Week）
- 文章与周记并存
- 支持标签与全文搜索
- 轻量可视化（可选）

更多设计与技术细节见：`docs/plan.md`。

## 本地运行
```bash
npm install
npm run dev
```

## 目录结构（简版）
- `app/`：页面与路由
- `data/`：周记与文章的示例数据
- `components/`：基础 UI 组件
- `lib/`：数据聚合与搜索逻辑
